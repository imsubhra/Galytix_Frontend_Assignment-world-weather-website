## Instructions

1. `git clone https://github.com/imsubhra/Galytix_Frontend_Assignment-world-weather-website.git` 
2. `cd world_weather_website`
3. `npm install`
4. `npm run build`
5.  `ng serve`
6. `open your browser on http://localhost:4200/`
